## הגדרת Tasker לשליחת SMS על חבילות לבוט הטלגרם

מדריך זה יסביר כיצד להגדיר את אפליקציית Tasker באנדרואיד כדי שתעביר באופן אוטומטי כל הודעת SMS המכילה את המילה "חבילה" ישירות לבוט ניהול החבילות שלכם בטלגרם.

יש לבצע את ההגדרות הבאות על כל טלפון ממנו תרצו להעביר הודעות.

### דרישות מקדימות

1.  אפליקציית **Tasker** מותקנת על מכשיר האנדרואיד. (זוהי אפליקציה בתשלום).
2.  אפליקציית **טלגרם** מותקנת ומוגדרת.
3.  הבוט שיצרנו פועל וזמין.
4.  הוספתם את הבוט לקבוצת הטלגרם המשותפת שלכם.
5.  **מזהה הצ'אט (Chat ID)** של הקבוצה. כדי להשיג אותו:
    *   הריצו את הפקודה `/start` בקבוצה בה נמצא הבוט.
    *   הבוט ישיב עם הודעת פתיחה, ובסופה יציין את מזהה הצ'אט. לדוגמה: `✅ הצ'אט הזה נרשם בהצלחה (ID: -123456789)`.
    *   שמרו את המספר הזה (כולל המינוס, אם קיים). זהו ה-`CHAT_ID` שלכם.

### שלבי הגדרה ב-Tasker

#### שלב 1: יצירת פרופיל חדש

1.  פתחו את Tasker.
2.  עברו ללשונית **Profiles**.
3.  לחצו על כפתור הפלוס (`+`) בתחתית המסך ובחרו **Event**.
4.  בחלון שנפתח, בחרו **Phone**.
5.  בחרו **Received SMS**.

#### שלב 2: הגדרת התנאי (Trigger)

במסך הגדרת ה-SMS, מלאו את הפרטים הבאים:

1.  **Type**: השאירו על `Any`.
2.  **Sender**: השאירו ריק כדי לקבל הודעות מכל שולח.
3.  **Content**: כתבו `*חבילה*`.
    *   הכוכביות (`*`) משני הצדדים גורמות ל-Tasker לחפש את המילה "חבילה" בכל מקום בתוך treść הודעת ה-SMS.
4.  לחצו על כפתור החזור (Back) למעלה כדי לשמור.

#### שלב 3: יצירת משימה (Task) חדשה

1.  מיד לאחר שמירת הפרופיל, יקפוץ תפריט שיבקש מכם לקשר אליו משימה.
2.  בחרו **New Task**.
3.  תנו למשימה שם, למשל: `Forward Package SMS`.
4.  לחצו על כפתור הוי (`✓`).

#### שלב 4: הוספת פעולת שליחת ההודעה

בתוך מסך עריכת המשימה, נוסיף את הפעולה ששולחת את המידע לטלגרם.

1.  לחצו על כפתור הפלוס (`+`) בתחתית המסך כדי להוסיף פעולה.
2.  בחרו **Net**.
3.  בחרו **HTTP Request**.

#### שלב 5: הגדרת פרטי הבקשה (HTTP Request)

במסך הגדרת הפעולה, מלאו את השדות הבאים בקפידה:

1.  **Method**: בחרו `POST`.
2.  **URL**: הדביקו את הכתובת הבאה, והחליפו את `<YOUR_BOT_TOKEN>` בטוקן הבוט שלכם:
    ```
    https://api.telegram.org/bot8766003392:AAH1TmcrbuvTLBuul6DVsNhP6DnBgdtpdrM/sendMessage
    ```
3.  **Headers**: בשדה זה, כתבו:
    ```
    Content-Type: application/json
    ```
4.  **Body**: הדביקו את הטקסט הבא, והחליפו את `<YOUR_CHAT_ID>` במזהה הצ'אט ששמרתם קודם:
    ```json
    {
      "chat_id": "<YOUR_CHAT_ID>",
      "text": "%SMSRB\n\n(נשלח מ: %SMSRF)"
    }
    ```
    *   `%SMSRB` הוא משתנה של Tasker שמכיל את גוף הודעת ה-SMS שהתקבלה.
    *   `%SMSRF` הוא משתנה שמכיל את מספר הטלפון של שולח ה-SMS.

5.  **Timeout**: מומלץ להגדיר ל-`15` שניות.

#### שלב 6: שמירה והפעלה

1.  לחצו על כפתור החזור (Back) כדי לשמור את הפעולה.
2.  לחצו שוב על חזור כדי לצאת ממסך עריכת המשימה.
3.  ודאו שהפרופיל שיצרתם בלשונית **Profiles** מופעל (המתג שלידו צריך להיות ירוק/פעיל).

זהו! כעת, בכל פעם שתתקבל הודעת SMS המכילה את המילה "חבילה", Tasker יעביר אותה אוטומטית לקבוצת הטלגרם שלכם, והבוט יטפל בה.

**יש לחזור על כל השלבים במכשיר הטלפון השני.**
