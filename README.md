# בוט טלגרם לניהול חבילות

בוט טלגרם זה נועד לעזור בניהול ומעקב אחר חבילות המתקבלות באמצעות הודעות SMS. הבוט מאזין להודעות, שומר אותן, ושולח תזכורות תקופתיות לקבוצת טלגרם.

## תכונות

-   **קליטת הודעות SMS**: מקבל הודעות SMS המועברות אליו (למשל, דרך אפליקציית Tasker).
-   **מסד נתונים**: שומר את כל המידע על החבילות במסד נתונים מקומי (SQLite).
-   **תזכורות אוטומטיות**: שולח תזכורת כל 3 ימים על חבילות שטרם נאספו.
-   **כפתורי Inline**: מאפשר סימון מהיר של חבילה כ"נאספה" ישירות מהצ'אט.
-   **תמיכה בקבוצות**: עובד בסביבה קבוצתית כך שמספר משתמשים יכולים לנהל את החבילות.
-   **ניהול פשוט**: כולל פקודות להצגת חבילות ממתינות, כל החבילות, והוספה ידנית.

## דרישות מערכת

-   Python 3.7+
-   ספריית `python-telegram-bot` עם תמיכה ב-`job-queue`.

## הפעלת הבוט

ניתן להריץ את הבוט בשתי דרכים:

### 1. הפעלה ישירה (לצורך בדיקות)

כדי להפעיל את הבוט באופן ישיר מהטרמינל, נווט לתיקיית הפרויקט והרץ:

```bash
python3 bot.py
```

הבוט יתחיל לרוץ. שים לב שהפעלה זו תפסיק ברגע שתסגור את הטרמינל.

### 2. הפעלה כשירות רציף (מומלץ)

כדי להבטיח שהבוט ירוץ באופן קבוע, גם לאחר הפעלה מחדש של השרת, מומלץ להגדיר אותו כשירות `systemd`.

**א. יצירת קובץ שירות:**

צור קובץ חדש בנתיב `/etc/systemd/system/package_bot.service`:

```bash
sudo nano /etc/systemd/system/package_bot.service
```

הדבק את התוכן הבא לתוך הקובץ. **ודא שהנתיב לקובץ `bot.py` ב-`ExecStart` נכון**.

```ini
[Unit]
Description=Telegram Package Management Bot
After=network.target

[Service]
User=ubuntu
Group=ubuntu
WorkingDirectory=/home/ubuntu/package_bot
ExecStart=/usr/bin/python3 /home/ubuntu/package_bot/bot.py
Restart=always

[Install]
WantedBy=multi-user.target
```

**ב. הפעלת השירות:**

לאחר שמירת הקובץ, הרץ את הפקודות הבאות כדי להפעיל ולאפשר את השירות:

```bash
# טען מחדש את הגדרות systemd
sudo systemctl daemon-reload

# הפעל את השירות עכשיו
sudo systemctl start package_bot

# אפשר לשירות לעלות אוטומטית עם המערכת
sudo systemctl enable package_bot

# בדוק את סטטוס השירות
sudo systemctl status package_bot
```

## פקודות הבוט

-   `/start` - מאתחל את הבוט בצ'אט, רושם את הצ'אט לקבלת תזכורות ומציג הודעת פתיחה.
-   `/pending` - מציג את כל החבילות שטרם נאספו.
-   `/all` - מציג את 20 החבילות האחרונות (כולל אלו שנאספו).
-   `/add [תיאור]` - מוסיף חבילה חדשה באופן ידני.
-   `/remind` - שולח תזכורת מיידית על חבילות ממתינות.
-   `/stats` - מציג סטטיסטיקות על כמות החבילות.
-   `/help` - מציג הודעת עזרה.

## חיבור ל-Tasker

כדי שהבוט יקבל הודעות SMS באופן אוטומטי, יש להגדיר את אפליקציית Tasker באנדרואיד.

עקוב אחר ההוראות המפורטות בקובץ `tasker_instructions.md`.
